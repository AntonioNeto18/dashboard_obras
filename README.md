# 📊 Dashboard de Produtividade em Obras

Este projeto consiste em um **dashboard interativo desenvolvido em Python utilizando Streamlit** para análise de produtividade em obras da construção civil.

A aplicação permite explorar dados de execução de serviços, analisar indicadores de produtividade e identificar variações entre diferentes obras e grupos de atividades.

---

# 🚀 Tecnologias utilizadas

- Python
- Streamlit
- Pandas
- Plotly
- Excel (base de dados)


---

# 📊 Funcionalidades

O dashboard permite:

- 📈 Visualização de **indicadores de produtividade**
- 📊 Gráficos interativos
- 📋 Estatísticas descritivas
- 🔎 Comparação entre **obras e grupos de serviços**

---

# 📊 Análises disponíveis

O dashboard apresenta:

- Média
- Mediana
- Distribuição dos dados
- Boxplots de produtividade
- Comparação entre diferentes obras

Essas análises permitem identificar:

- Serviços mais produtivos
- Obras com melhor desempenho
- Variabilidade da produtividade

# ▶️ Como executar o projeto

### 1️⃣ Clone o repositório

```bash
git clone https://github.com/AntonioNeto18/dashboard_obras.git
```

### 2️⃣ Acesse a pasta do projeto
```bash
cd dashboard_obras
```

### 3️⃣ Instale as dependências
```bash
pip install -r requirements.txt
```

### 4️⃣ Execute o dashboard
```bash
streamlit run app.py
```
